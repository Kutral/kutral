## 🔥 Open Source — `VoxDrop`

<div align="center">

> **Free Wispr Flow alternative for Windows. Hold `Ctrl+Win`, speak, release. Done.**

[![VoxDrop](https://img.shields.io/badge/VoxDrop-🎙%20Voice%20Dictation-FF3B30?style=for-the-badge)](https://github.com/Kutral/VoxDrop)
[![MIT License](https://img.shields.io/badge/License-MIT-0b1220?style=for-the-badge)](https://github.com/Kutral/VoxDrop/blob/main/LICENSE)
[![Stars](https://img.shields.io/github/stars/Kutral/VoxDrop?style=for-the-badge&color=f1c40f)](https://github.com/Kutral/VoxDrop/stargazers)

</div>

**VoxDrop** is a hotkey-powered desktop dictation app that captures your voice, cleans it up with Groq AI, and pastes polished text into any app — instantly. No subscription. No trial. Free forever.

| Feature | What it does |
| :--- | :--- |
| 🎙️ **Hold-to-speak** | Global `Ctrl+Win` hotkey, works in any app |
| 🧠 **AI Cleanup** | Removes fillers, adds punctuation, handles backtracks via Groq Llama |
| ✂️ **Snippets** | Say a trigger → expands to your full canned text |
| 📋 **History** | Every transcription saved locally with search |
| ⚡ **Lightweight** | ~50MB RAM. Lives in your system tray. |

```
Stack: Tauri 2.0 · Rust · React 18 · TypeScript · Groq Whisper + Llama · SQLite
```

**→ [github.com/Kutral/VoxDrop](https://github.com/Kutral/VoxDrop)** — Star it if you find it useful ⭐
